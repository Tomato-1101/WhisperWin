# SuperWhisper-like Tool

OpenAI Whisperを使用した、常駐型音声入力ツールです。
ホットキーを押すだけで録音を開始し、もう一度押すと文字起こしをしてアクティブなウィンドウに入力します。

## 特徴

- **グローバルホットキー**: どのアプリを使っていても、設定したキー（デフォルト: F2）で起動。
- **高精度文字起こし**: OpenAI Whisperモデルを使用。
- **自動入力**: 文字起こし結果を自動的に貼り付け。
- **設定可能**: モデルサイズ、ホットキー、言語などを `settings.yaml` で簡単に変更可能。

## インストール

1. 必要なライブラリをインストールします。
   ```bash
   pip install -r requirements.txt
   ```
   ※ GPUを使用する場合は、PyTorchのCUDA対応版がインストールされていることを確認してください。

2. `ffmpeg` がシステムにインストールされている必要があります。

## 使い方

1. アプリケーションを起動します。
   ```bash
   python src/main.py
   ```
   初回起動時はモデルのダウンロードが行われるため、時間がかかります。

2. 「Ready! Press 'f2' to toggle recording.」と表示されたら準備完了です。

3. **F2キー**（または設定したキー）を押すと録音が開始されます。
   - コンソールに `[Start Recording]` と表示されます。

4. 話し終わったら、もう一度 **F2キー** を押します。
   - 録音が停止し、文字起こしが始まります。
   - 完了すると、アクティブなウィンドウ（メモ帳やブラウザなど）にテキストが入力されます。

## 設定

`settings.yaml` を編集してカスタマイズできます。

```yaml
hotkey: 'f2'          # 起動キー (例: 'f2', 'ctrl+space')
model_size: 'base'    # モデルサイズ (tiny, base, small, medium, large)
device: 'cuda'        # 'cuda' (GPU) または 'cpu'
language: 'ja'        # 言語コード (ja, en など)
```

## トラブルシューティング

- **入力されない**: 管理者権限で実行されているアプリに入力する場合、このツールも管理者権限で実行する必要がある場合があります。
- **エラーが出る**: `ffmpeg` がパスに通っているか確認してください。
